# ChatGPTVoiceMode
Enhanced Dualistic Communication Potentials

1. Download and install the supplied Android APK.
2. Open the application and leave the mode set to LOCAL_ONLY.
3. Select each preset conversational scenario.
4. Press “Analyse ASR Packet.”
5. Observe the Pertinence, Prevalence, Precedence, and Confidence scores.
6. Compare the selected response-gate action and state transition.
7. Enter custom partial speech text to test additional cases.
8. API_ASSISTED mode is optional and falls back safely when unavailable.
